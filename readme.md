# Crawl Story
Crawl story là ứng dụng nodejs dùng để crawl truyện từ truyenchon.com, hamtruyen.com và hocvientruyentranh.net. Đây chỉ là bản demo và chưa hoàn thiện.
## Cài đặt

    $ https://github.com/duonghuan2122000/crawl-story.git
    $ cd crawl-story
    $ npm install
    $ npm start
## Lưu ý
Đây chỉ là bản demo.
